"LiDAR Data Filtering and DTM Generation using Empirical Mode Decomposition"
(Matlab code for iterative morphology based EMD algorithm for detecting objects and generating DTM from LiDAR Data)

*demo1.m: ISPRS data set example (default data: 'samp11' )
	If you want to try another sample, just comment out current 'fileName' (line number 21 in the code) and open another sample (line number 22 or 23 in the code)

*demo2.m: NSF data set example for residential area (utah-1)

*demo2.m: NSF data set example for forest area (andrews-1)
